# OOP Chess Game in C++

A console-based Chess Game developed in C++ using Object-Oriented Programming (OOP) concepts.
This project demonstrates the implementation of OOP principles by designing chess pieces, game logic, and player interactions in a structured way.

---

## Project Overview

The **OOP Chess Game** is a console application where two players can play a chess match through a command-line interface.
The main purpose of this project is to apply Object-Oriented Programming concepts in a real-world application by creating different chess pieces as objects and managing their behavior using classes.

---

## Features

* Complete chess board representation
* Two-player gameplay
* Different chess pieces implementation
* Valid move handling
* Turn-based gameplay system
* Console-based user interface
* Structured object-oriented design

---

## Technologies Used

* Programming Language: C++
* Concepts: Object-Oriented Programming (OOP)
* IDE: Microsoft Visual Studio 2015

---

## OOP Concepts Implemented

This project demonstrates the following OOP concepts:

### Classes and Objects

* Each chess piece is represented as an object with its own properties and behaviors.

### Encapsulation

* Data and functions are grouped together inside classes to maintain proper structure.

### Inheritance

* Common properties and behaviors are shared among different chess pieces.

### Polymorphism

* Different chess pieces implement their movement behavior in their own way.

### Abstraction

* Complex game logic is hidden behind simple class interfaces.
